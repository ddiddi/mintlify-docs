## Introduction

### What is Solo?

Solo is an open-source platform for **Hardware Aware Inference**.&#x20;

Using Solo, you can easily set up your environment, download and serve models, benchmark performance, fine-tune, ensemble, distill, and deploy across a variety of inference engines and platforms.&#x20;

## Prerequisites

Before installing Solo Server, ensure your system meets the following prerequisites:

### Software Requirements

* **Docker:**




  Solo Server is designed to run within a containerized environment. Make sure Docker is installed on your system.

  * [Docker Installation Guide](https://www.docker.com/get-started/)

### Hardware Requirements

* **Processor:** At least a<u> 1 GHz CPU core</u> (more cores are recommended for production workloads).

* **Memory:** Minimum of <u>2GB RAM</u> (4GB or more is recommended for smoother performance, especially under load).

* **Storage:** 

### Operating System Requirements

* **Linux:** Ubuntu 18.04 or later, CentOS 7+, or equivalent distributions.

* **macOS:** Version 10.13 or later.

* **Windows:** Windows 10 or later (Docker Desktop required).

These prerequisites ensure that Solo Server runs efficiently in your environment while leveraging containerization for consistent deployment.
